# AITesterBluePrint2X 🚀

Welcome to the **AITesterBluePrint2X** repository. This is a comprehensive roadmap and collection of projects focused on **AI-Driven Software Testing**, **Prompt Engineering**, and **Automated Testing Ecosystems**.

## 📂 Repository Structure

This repository is organized into chapters and specialized modules covering everything from local LLM integration to advanced browser automation.

### 📍 Chapter 01: Local LLM & Test Generation
- **[Project 01: Local LLM Test Generator](./Chapter_01/Project01_LocalLLM_TestGenerator)**: Harnessing local language models (like Llama/DeepSeek) specifically for automated test case generation.
- **[Project 01 (VSCode Edition)](./Chapter_01/Project01_LocalLLM_TestGenerator_VSCode)**: Visual Studio Code integrated local LLM tools for test development.

### 📍 Chapter 02: Prompt Engineering & Frameworks
- **[Project 01: RICEPOT Prompt Selenium FW](./Chapter_02/Project_01_RICEPOT_Prompt_Selenium_FW)**: Combining RICEPOT prompt engineering framework with traditional Selenium Java automation.
- **[Project 02: Prompt Templates](./Chapter_02/Project_02_Prompt_Templates)**: Curated templates for enterprise-level automation prompts.
- **[Project 02 (Real-World Application)](./Chapter_02/Project_02_Real_PromptEngg)**: Advanced techniques in prompt chaining and context management for QA.

### 📍 Chapter 03: Modern Web Apps & Career Management
- **[Project 01: Job Tracker (JobFlow)](./Chaptor_03/Project_01_Job_Tracker)**: A local-first, premium **Job Tracker** built with **React 18**, **Vite**, **Tailwind CSS v4**, and **IndexedDB**. Features professional Kanban boards with drag-and-drop support.
- **Java Concepts Series**: In-depth tutorials covering core Java logic, from basics to functional programming (Chapter 01 - 10).

### 📍 Specialized Modules
- **[PlayWright Automation](./PlayWright)**: Modern browser testing using the Playwright framework, featuring enterprise-standard configurations.
- **[PlayWright MCP (Antigravity Edition)](./PlayWrigt_MCP)**: Advanced AI-agentic automation using the Model Context Protocol (MCP) for autonomous browser testing and report generation.

---

## 🛠️ Technology Stack
- **Frameworks**: React, Playwright, Selenium, TestNG
- **Languages**: JavaScript (Vite), Java, TypeScript
- **AI/LLM**: Local-first LLMs, Prompt Engineering, MCP
- **Database**: IndexedDB (Local-First Persistence)
- **Styling**: Tailwind CSS v4, Vanilla CSS (Premium Aesthetics)

## 📌 Getting Started
Most projects in this repository are standalone. For web-based projects (like Job Tracker):
1. Navigate to the project folder.
2. Run `npm install`.
3. Run `npm run dev`.
